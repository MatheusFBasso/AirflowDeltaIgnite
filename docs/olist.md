# Olist Pipeline Documentation

## Overview

The Olist pipeline processes the [Olist Brazilian E-Commerce Public Dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) using the Medallion architecture. Olist is a dataset covering 100k orders from 2016-2018, including customers, products, payments, reviews, and logistics. The pipeline ingests data (via API or local files), transforms it through Bronze (raw partitioned), Silver (cleaned/joined with quality checks), and Gold (aggregated metrics like delivery times, seller performance), storing results in Delta Tables for ACID compliance and time travel.

- **Focus**: E-commerce analytics (e.g., order fulfillment, customer satisfaction, revenue trends).
- **DAGs**: `dags/olist/` orchestrates ingestion, transformations, and quality checks.
- **ETL Code**: `etl/olist/` handles API fetching and PySpark/Delta processing.
- **Configs**: `config/olist_config.yaml` (API endpoints, Delta paths, quality thresholds).
- **Data Location**: `/opt/spark/data/warehouse/` (bronze/silver/gold in `warehouse.db`).

---

## Pipeline Flow

1. **Ingestion DAG** (`ingestion_dag.py`): Fetches data via `olist_api.py`, lands in raw, moves to bronze via `bronze.py`.
2. **Bronze to Silver DAG** (`bronze_to_silver_dag.py`): Cleans/joins with `bronze_to_delta.py`, runs quality checks.
3. **Silver to Gold DAG** (`silver_to_gold_dag.py`): Aggregates with `silver_to_delta.py` and `gold_to_delta.py`, generates metrics.
4. **Data Quality DAG** (`data_quality_dag.py`): Validates across layers using `data_quality.py`.

All DAGs use `SparkSubmitOperator` with `spark_conn` (client mode, executor_cores=1, total_executor_cores=2) and `verbose=False` for minimal logs.

## Extract (Ingestion)

**DAG**: `ingestion_dag.py` – Schedules API calls or file loads to raw layer.

**ETL**: `etl/olist/apis/olist_api.py` – Fetches datasets (orders, payments, etc.) using `requests`.

**Key Steps**:
- API call to Olist endpoints (e.g., `/orders`, `/payments`).
- Save as JSON/CSV to `/opt/spark/data/warehouse/raw_data/olist/`.
- BashOperator moves to bronze with timestamp (using `get_bash_command`).

**Example Code Snippet** (from `olist_api.py`):
```python
import requests
from pyspark.sql import SparkSession

def fetch_olist_orders(spark: SparkSession):
    response = requests.get("https://api.olist.com/orders")  # Mock endpoint
    df = spark.createDataFrame(response.json())
    df.write.format("delta").mode("append").save("/opt/spark/data/warehouse/raw_data/olist/orders")
```

**Output**: Raw Delta tables partitioned by date (e.g., `olist_orders_dataset`).

## Bronze Layer

**DAG**: `bronze_to_delta_dag.py` – Partitions raw data into Delta Tables.

**ETL**: `etl/olist/transformations/bronze.py` & `bronze_to_delta.py` – Basic partitioning, schema inference.

**Key Steps**:
- Read from raw: `spark.read.format("delta").load("/opt/spark/data/warehouse/raw_data/olist/")`.
- Partition by key (e.g., `order_date`).
- Write to `/opt/spark/data/warehouse/bronze.db/olist_orders_dataset` with Delta configs.

**Example Code Snippet** (from `bronze_to_delta.py`):
```python
from etl.common.utils import get_spark_session

spark = get_spark_session("OlistBronze")
df = spark.read.format("delta").load("/opt/spark/data/warehouse/raw_data/olist/orders")
df.write.format("delta").partitionBy("order_date").mode("overwrite").save("/opt/spark/data/warehouse/bronze.db/olist_orders_dataset")
```

**Quality Checks**: Null counts on primary keys (e.g., `order_id`).

**Output**: Partitioned Delta Tables (e.g., `olist_orders_dataset`, `olist_payments_dataset`).

## Silver Layer

**DAG**: `bronze_to_silver_dag.py` – Joins datasets, cleans, validates.

**ETL**: `etl/olist/transformations/silver.py` & `bronze_to_delta.py` – Dedup, type casting, joins.

**Key Steps**:
- Join bronze tables (e.g., orders + payments on `order_id`).
- Apply transformations (e.g., `df.withColumn("payment_total", col("payment_value").cast("double"))`).
- Quality: `check_data_quality(df, columns=["order_id"], check_nulls=True, check_uniques=True)`.
- Write to `/opt/spark/data/warehouse/silver.db/olist_orders_dataset`.

**Example Code Snippet** (from `silver.py`):
```python
from etl.common.utils import get_spark_session, check_data_quality

spark = get_spark_session("OlistSilver")
orders_df = spark.read.format("delta").load("/opt/spark/data/warehouse/bronze.db/olist_orders_dataset")
payments_df = spark.read.format("delta").load("/opt/spark/data/warehouse/bronze.db/olist_payments_dataset")
joined_df = orders_df.join(payments_df, "order_id")
quality_report = check_data_quality(joined_df, ["order_id"])
if quality_report['nulls']['order_id'] > 0:
    raise ValueError("Quality check failed")
joined_df.write.format("delta").mode("overwrite").save("/opt/spark/data/warehouse/silver.db/olist_orders_dataset")
```

**Output**: Cleaned, joined Delta Tables (e.g., `olist_order_payments_dataset`).

## Gold Layer

**DAG**: `silver_to_gold_dag.py` – Aggregates for business insights.

**ETL**: `etl/olist/transformations/gold.py` & `gold_to_delta.py` – Aggregations, metrics.

**Key Steps**:
- Aggregate silver data (e.g., `groupBy("seller_id").agg(avg("delivery_time").as("avg_delivery"))`).
- Compute KPIs (e.g., seller performance, revenue trends).
- Write to `/opt/spark/data/warehouse/gold.db/olist_delivery_time_table` or `olist_sellers_performance`.

**Example Code Snippet** (from `gold_to_delta.py`):
```python
from etl.common.utils import get_spark_session

spark = get_spark_session("OlistGold")
silver_df = spark.read.format("delta").load("/opt/spark/data/warehouse/silver.db/olist_orders_dataset")
gold_df = silver_df.groupBy("seller_id").agg(
    avg("delivery_time").alias("avg_delivery_days"),
    count("order_id").alias("total_orders")
).orderBy(col("avg_delivery_days"))
gold_df.write.format("delta").mode("overwrite").save("/opt/spark/data/warehouse/gold.db/olist_sellers_performance")
```
